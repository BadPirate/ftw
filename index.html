Frosty is an iOS Bitcoin wallet designed to be "Trustless"
It is designed to allow for signing and sending of transactions
using only QR Codes, so that it does not use the internet or require
a direct connection to a computer. For truly trustless storage of
your valuable private keys, turn off the internet on the phone you install
Frosty on after downloading.
